# FORMA — Инструкция по загрузке ресурсов

## Порядок загрузки

1. Откройте http://localhost:5173/become-author и подайте заявку на статус автора
2. В базе данных обновите статус: UPDATE authors SET verification_status='VERIFIED' WHERE id=1;
3. Перелогиньтесь (выйти и войти снова)
4. Перейдите на http://localhost:5173/resources/upload
5. Для каждого ресурса ниже заполните форму
